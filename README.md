<h1 align="center">🍷 Montevino – Restaurant Web Application (Frontend)</h1>

<p align="center">
Modern frontend for a full stack restaurant platform featuring reservations, dynamic menus, and payment integration.
</p>


<h2 align="center">🧠 Project Overview</h2>

<p>
Montevino is a <b>full stack restaurant application</b> designed to provide users with a seamless experience when browsing menus, making reservations, and managing orders.
</p>

<p>
This repository contains the <b>frontend</b> of the application, built with modern technologies and focused on performance, user experience, and scalability.
</p>


<h2 align="center">⚡ Key Features</h2>

<ul>
  <li>🍽️ Dynamic menu with categories (food & drinks)</li>
  <li>🔎 Filters (vegetarian, gluten-free, etc.)</li>
  <li>📅 Reservation system (date, time, number of people)</li>
  <li>🧾 Order selection (dishes & beverages)</li>
  <li>💳 Payment flow integration (deposit + full payment)</li>
  <li>👤 User dashboard (profile & reservations)</li>
  <li>🛠️ Admin dashboard (reservation management)</li>
</ul>

<h2 align="center">🚀 Getting Started</h2>

<h3 align="center">1. Clone the repository</h3>

<pre>
git clone https://github.com/kyara25e/MontevinoFront.git
</pre>

<h3 align="center">2. Install dependencies</h3>

<pre>
npm install
</pre>

<h3 align="center">3. Run the development server</h3>

<pre>
npm run dev
</pre>


<h2 align="center">🔐 Environment Variables</h2>

<pre>
NEXT_PUBLIC_API_URL=your_backend_url
</pre>
